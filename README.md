## vleet

**vleet is a terminal-only CLI that lets you solve LeetCode problems in Vim**:

- Fetch the full problem statement + metadata from LeetCode
- Generate a local workspace with the official starter snippet for your chosen language
- Open `vim` (or `$EDITOR`) to write code
- Submit to LeetCode and show the final verdict + details
